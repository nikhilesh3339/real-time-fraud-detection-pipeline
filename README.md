# Real-Time Fraud Detection Pipeline

This project simulates a real-time fraud detection system using Kafka, Spark Structured Streaming, Airflow, dbt, and Snowflake.

The goal of this project is to show how streaming transaction data can be ingested, processed in real time, and transformed into analytics-ready datasets for fraud monitoring.

---

## 🚀 What this project does

- Simulates transaction events using Kafka
- Processes streaming data using Spark Structured Streaming
- Applies simple fraud detection logic based on transaction patterns
- Stores processed data in Snowflake
- Uses dbt to build clean, analytics-ready models
- Orchestrates workflows using Airflow

---

## 🏗️ Architecture

Kafka → Spark Streaming → Snowflake → dbt → Analytics

---

## 🛠️ Tech Stack

- Kafka
- Spark Structured Streaming
- Airflow
- dbt
- Snowflake
- Docker
